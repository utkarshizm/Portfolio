Utkarsh Pandey | AI & Data Science Portfolio

Welcome to my personal portfolio website! This is a fully responsive, dark-themed, interactive web application built from scratch to showcase my projects, skills, and experience in AI, Data Science, and Business Intelligence.

🌐 Live Website: utkarshizm.github.io/Portfolio

✨ Features

  Interactive 3D Background: Built with Three.js, featuring a dynamic particle system and wireframe geometry.
  GSAP Scroll Animations: Smooth scroll-triggered reveal animations, scroll progress bars, and staggered element loading.
  Customizable Cursor Engine: A built-from-scratch interactive cursor system with 10 different modes (Minimalist, Blend Mode, Comet Trail, Tech Crosshair, Spotlight, etc.) including a settings widget.    3D Tilt Effect: Project cards respond to mouse movement with a 3D perspective transform.
  Fully Responsive: Mobile-first design with a custom slide-in navigation menu for touch devices.

🛠️ Tech Stack

  Frontend: HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript (ES6)
  Animation: GSAP (GreenSock) with ScrollTrigger
  3D Graphics: Three.js
  Icons & Fonts: Font Awesome, Google Fonts (Syne & Inter)
